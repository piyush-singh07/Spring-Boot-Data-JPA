* A demo Application for getting a basic idea on how to use Spring Data JPA with Spring Boot

* Currently there is no exception handling and other secondary business logic as its just a demo to 
  get a gist how exactly Data JPA can be integrated with Spring Boot.
 
* Inmemory Database [H2 Database] is used.

* Components Used:

	-> STS
	-> https://start.spring.io [For getting a starter project]
	-> Spring Web
	-> Spring Data JPA
	-> H2 Database 
	-> Project Lombok
	-> Dev Tools
